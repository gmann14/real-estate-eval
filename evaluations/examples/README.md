# Examples

Synthetic worked examples, safe to commit and share publicly.

> **These are NOT real properties.** Addresses, agent names, financial
> figures, and seller histories in these examples are fictional. They exist
> to illustrate how the analysis tool works and to serve as regression
> fixtures.

## Contributing an example

Follow the structure of an existing example. Required files:

- `input.md` — filled-in evaluation template
- `analysis.md` — full analysis report
- `enhancements.md` — ROI enhancement recommendations

Optional:

- `email-to-realtor.md` — sample diligence questions
- `notes.md` — walkthrough of what the example demonstrates

**Every file must carry a "synthetic example" banner at the top.** No real
addresses, no real agent names, no real phone numbers, no confidential
seller data.
